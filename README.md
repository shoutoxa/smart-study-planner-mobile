# Smart Study Planner Mobile

A mobile application built with **React Native** and **Expo** to help users schedule, plan, and manage their study sessions efficiently.

## 🚀 Features

- **Study Planning:** Organize and schedule study sessions seamlessly.
- **Local Database:** Uses SQLite for fast, offline, on-device data storage.
- **Charts & Statistics:** Visualize your study progress with interactive charts (via `react-native-chart-kit`).
- **Modern UI:** Styled with **NativeWind** (Tailwind CSS) for clean and responsive interfaces.
- **Navigation:** Deep and organized routing powered by React Navigation (native stack and bottom tabs).

## 🛠️ Tech Stack

- [React Native](https://reactnative.dev/) & [Expo](https://expo.dev/)
- [React Navigation](https://reactnavigation.org/)
- [NativeWind](https://www.nativewind.dev/)
- [Expo SQLite](https://docs.expo.dev/versions/latest/sdk/sqlite/)
- [React Native Chart Kit](https://github.com/indiespirit/react-native-chart-kit)

## 📦 Installation & Setup

1. **Clone the repository** (if uploaded to GitHub):

   ```bash
   git clone https://github.com/your-username/smart_study_planner_mobile.git
   ```

2. **Navigate to the project folder:**

   ```bash
   cd smart_study_planner_mobile
   ```

3. **Install dependencies:**
   Make sure you have Node.js installed, then run:

   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm start
   ```

## 📱 Running on Device / Emulator

- **Android:** Press `a` in the Expo terminal or run `npm run android`
- **iOS:** Press `i` in the Expo terminal or run `npm run ios`

You can also use the **Expo Go** app on your physical device to scan the QR code generated by the development server.

## 🚀 Building for Production

This app is configured to use **EAS (Expo Application Services)** for building production-ready apps.

To build an Android APK:

```bash
eas build -p android --profile production
```
